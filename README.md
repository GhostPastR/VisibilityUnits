# VisibilityUnits
bin.zip - архив с .exe файлами для запуска приложения
ConsoleCalcVisibility.exe - запускает в фоновом режиме алгоритм расчёта.
Gui.exe - запускают gui версию алгоритма расчёта.
Призапуске необходим файл data_units.txt с исходными даннами.
Результат работы бцдет файл result.txt.
Алгоритм расчета находится в классе ListUnits.
